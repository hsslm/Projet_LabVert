LabVert - Système de surveillance de plantes connecté
Système IoT de surveillance de plantes en temps réel. L'ESP32 collecte la température et l'humidité via un capteur DHT11 et envoie les données à une API Node.js qui les stocke dans MongoDB Atlas. Un dashboard web affiche les mesures en direct avec graphiques et statistiques.
Pour plus de détails : https://hsslm.github.io/Projet_LabVert/

LabVert - Connected Plant Monitoring System
A real-time IoT plant monitoring system. The ESP32 collects temperature and humidity data via a DHT11 sensor and sends it to a Node.js API that stores it in MongoDB Atlas. A web dashboard displays live readings with charts and statistics.
More details : https://hsslm.github.io/Projet_LabVert/
