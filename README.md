LabVert est un système d'arrosage intelligent connecté permettant de surveiller en temps réel l'état d'une plante et de gérer son arrosage de manière optimale. Le système offre deux modes de fonctionnement : un mode manuel où l'utilisateur contrôle directement l'arrosage via une interface web, et un mode automatique où une intelligence artificielle analyse les données des capteurs et prend des décisions d'arrosage en fonction des besoins spécifiques de la plante
